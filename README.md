# 🎬 Movie Recommendation System

## 📌 Project Overview
This project implements a Movie Recommendation System using Machine Learning and K-Means Clustering. The system groups similar movies based on their genres and features, then generates personalized recommendations according to user preferences such as movie ratings and release year.

## 🎯 Objectives
- Recommend movies based on user preferences.
- Group similar movies using clustering techniques.
- Improve movie discovery through data-driven recommendations.
- Apply unsupervised learning to analyze movie patterns.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- K-Means Clustering

## 📂 Dataset Features
The dataset contains movie-related information such as:
- Movie Title
- Genres
- Rating
- Release Year
- Popularity Metrics

## ⚙️ Methodology

### 1. Data Preprocessing
- Data Cleaning
- Handling Missing Values
- Genre-based Feature Extraction
- One-Hot Encoding of Genres
- Feature Scaling using StandardScaler

### 2. Machine Learning
- Applied K-Means Clustering to group similar movies.
- Determined movie similarity based on encoded genre features and other attributes.
- Generated clusters representing different movie categories.

### 3. Recommendation Engine
- Accepts user preferences such as:
  - Preferred Rating
  - Release Year
- Identifies relevant movie clusters.
- Recommends movies with similar characteristics.

## 📊 Key Features
- Personalized Movie Recommendations
- Genre-Based Similarity Analysis
- Unsupervised Learning Approach
- Data Preprocessing and Feature Engineering
- Scalable Recommendation Framework

## 📈 Results
- Successfully grouped similar movies into clusters.
- Generated meaningful recommendations based on user preferences.
- Improved movie discovery using machine learning techniques.
